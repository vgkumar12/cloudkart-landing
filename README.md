# CloudKart Premium Marketing Website

Complete multi-page marketing website for CloudKart e-commerce platform.

## Pages Created

1. **Home** (`index.html`) - Hero, features, pricing preview
2. **Features** (`features.html`) - All 50+ features detailed
3. **Pricing** (`pricing.html`) - 3 pricing tiers with comparison
4. **Demo** (`demo.html`) - Video demos and screenshots
5. **About** (`about.html`) - Mission, tech stack, values
6. **Contact** (`contact.html`) - Contact information

## Design
- Purple theme from CloudKart logo (#6B46C1)
- Fully responsive
- Modern animations
- Professional layout
